В теге form прописывается атрибут script, он отвечает за адресс отправки заполненных строк.
Атрибут - method со значением GET позволяет видеть передаваемые данные, значение POST - не видеть
С помощью атрибута placeholder - можем задать подсказку прямо в поле ввода
С помощью атрибута required - можно сделать поля обязательным для заполниния
Атрибут readonly - делает поле невозможным для редактирования
type = "radio"
Радио кнопки, позволяют сдела графу с выбором элементов в виде точек, чтобы иметь возможность выбрать только одну кнопку, необходимо прописать каждому элементы одинаковое имя.
Атрибут checked - задает определенное значение выбранным по умолчанию
type="checkbox" - можно сделать несколько вариантов ответов
select multiple - позволяет через shift выбрать несколько значений
